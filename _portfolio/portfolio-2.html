Motivation

Parasites influence host health, behavior, and ecology, and understanding how parasites are shared across species is important for both conservation biology and disease ecology. Evolutionary history is thought to play a major role: closely related hosts tend to occupy similar ecological niches, share aspects of physiology, and therefore may be more likely to host similar parasite communities.

This analysis explores whether evolutionary divergence time and other ecological differences, predict the similarity of parasite communities between pairs of primate species.

The dataset is from Cooper et al. (2012), Ecology Letters, which examined host specificity and parasite sharing across primates.

Data

Each row represents a pair of primate species, described by:

  • divergence_time – millions of years since the species diverged.
  • distance – geographic distance (km) between species’ range centroids.
  • BMdiff – difference in body mass (grams).
  • precdiff – difference in annual precipitation (mm) across ranges.
  • parsim – parasite similarity, the proportion of parasites shared (0–1).

Exploratory Data Analysis

We begin by examining the relationship between divergence time and parasite similarity.

Code 
ggplot(parasites,aes(x=divergence_time,y=parsim))+
geom_point() +
labs(
title = "Parasite Similarity vs. Divergence Time",
x = "Divergence Time (millions of years)",
y = "Parasite Similarity"
)

Parasite similarity is inversely proportional to divergence time. Essentially, more recently diverged species have a greater degree of parasite similarity.

Modelling

Now will model this relationship with simple linear regression.

Code 
model = lm(parsim ~ divergence_time, data = parasites)
tidy(model)
# A tibble: 2 × 5
  term            estimate std.error statistic   p.value
  <chr>              <dbl>     <dbl>     <dbl>     <dbl>
1 (Intercept)      0.379    0.0121        31.4 4.59e-128
2 divergence_time -0.00653  0.000312     -20.9 2.70e- 73

The fitted equation is: \[ \hat{y} = 0.379487 -0.006532 \times divergence\_time\]

Code 
ggplot(parasites, aes(x = divergence_time, y = parsim)) +
geom_point() +
geom_smooth(method = "lm", se = FALSE) +
labs(
title = "Linear Model: Parasite Similarity vs. Divergence Time",
x = "Divergence Time (millions of years)",
y = "Parasite Similarity"
)

The model predicts a negative parasite similarity for species with very large divergence times, which is nonsensical. Additionally, parasim ranges between 0 and 1, which violates one of the assumptions required for linear regression. Thus, we need to transform the data.

Code 
parasites = mutate(parasites, transformed_parsim=log(parsim/(1-parsim)))

ggplot(parasites, aes(x = divergence_time, y = transformed_parsim)) +
geom_point() +
geom_smooth(method = "lm", se = FALSE) +
labs(
title = "Log-Transformed Parasite Similarity vs. Divergence Time"
)

The transformed relationship is more appropriate for linear regression: predictions now remain within valid bounds once back-transformed.

Now we will determine which variable is the strongest individual predictor of parasite similarity between species by first fitting a linear regression model to each pair of variables.

Code 
dt_model = lm(transformed_parsim ~ divergence_time, data = parasites)
dist_model = lm(transformed_parsim ~ distance, data = parasites)
BM_model = lm(transformed_parsim ~ BMdiff, data = parasites)
prec_model = lm(transformed_parsim ~ precdiff, data = parasites)

Explanatory Power of the models

Code 
tibble(
model = c("divergence_time", "distance", "BMdiff", "precdiff"),

Divergence time explains the most variability in parasite similarity, followed by geographic distance. Differences in body mass or precipitation explain very little by themselves.

Note: There is a significant correlation between distance and divergence time. That is expected since older splits usually mean species have moved farther apart. This overlap helps explain why both variables look like good predictors.

Conclusion

Evolutionary divergence time is the strongest single predictor of parasite sharing among primate species, explaining roughly half of the variation in parasite similarity. This aligns with biological expectations: closely related species tend to share ecological traits, physiology, and environmental exposure, facilitating similar parasite assemblages.

However, parasite similarity is a complex trait influenced by ecology, behavior, biogeography, and host defenses. Future work could extend this analysis using techniques such as multivariate models and considering the possiblity of nonlinear relationships.
